# 社区医院管理系统 java895

## 项目概述

社区医院管理系统是一款基于Spring Boot和Vue.js技术栈开发的软件，旨在为医院提供全面的信息化管理解决方案。该系统涵盖了患者、医生和管理员三个角色，以实现医院内部的高效运作和优质服务。

## 功能模块

### 管理员

- **管理员管理**：负责创建、管理用户账号，分配权限，进行用户角色设置。
- **基础数据管理**：管理医院运营所需的基础数据，如药品信息、设备信息、科室设置等。
- **药品管理**：负责药品的库存管理，包括药品采购、入库、出库、盘点等。
- **药品使用记录**：记录药品的使用情况，确保药品的合理使用和避免浪费。

### 医生角色

- **病例信息管理**：查阅、更新和记录患者的病例信息，包括病史、诊断结果、治疗方案等。
- **健康档案管理**：查看和更新患者的健康档案，包括健康状况、体检结果、疫苗接种等信息。
- **就诊信息管理**：处理患者的就诊流程，如挂号、就诊、开具处方、医嘱等。
- **医生管理**：接受系统的排班安排，参与绩效考核，管理自己的工作计划和日程。
- **就诊信息管理**：协助患者完成就诊流程，如缴费、取药等，确保流程顺畅。
- **患者信息核对**：核对患者信息，确保信息的准确性和完整性。

### 患者角色

- **用户管理**：注册、更新个人信息，设置账户权限（如查看病历、预约挂号等）。
- **健康档案管理**：查看自己的健康档案，了解健康状况和历史记录。
- **就诊信息管理**：预约挂号、查看就诊记录、缴费、取药等。
